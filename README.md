# Pomodoro-Buddy-Hackathon-Submission
PomodoroBuddy is a Pomodoro-based productivity app designed to improve focus, consistency, and sustainable work habits.

# TO RUN THE CODE 

# STEP 1- 
To the .env file add - VITE_SUPABASE_PROJECT_ID , VITE_SUPABASE_PUBLISHABLE_KEY and the VITE_SUPABASE_URL

# STEP 2- 
run (in cmd) - 
npm install      - to install all the dependencies.

# STEP 3 (final step!) - 
run (in cmd) - 
npm run dev        - to run the code in the browser
